_A system is in thermodynamic equilibrium_ **when nothing inside it wants to change anymore.**

`All measurable properties stay constant with time.`

Classical thermodynamics **does NOT describe a fully moving, changing system all at once.**

Because this is where system doesn't have constant [[Thermodynamic properties]]

**the properties are variables** when the system is running, and only static when we take a tiny part of time and use the classical thermodynamics. 

---
That Equilibrium only happens when **three equilibria exist at the same time**:

1. **Thermal equilibrium**  
    *Same temperature everywhere.*  
    No heat flow inside the system.
    
2. **Mechanical equilibrium**  
    *No unbalanced forces.*  
    Pressure is uniform. Nothing is moving or expanding.
    
3. **Chemical equilibrium**  
    *No net chemical reactions or diffusion.*  
    Composition is stable.
    

When all three are satisfied → **thermodynamic equilibrium.**

##### Thermodynamic equilibrium is the state in which a system has no driving forces for change and all macroscopic properties remain constant with time.
