## State

 A **state** is the condition of _a system described_ by a set of measurable _properties_ at a given instant.

If pressure, temperature, volume, etc. are fixed, the system is in a definite state.  
```Change any one → state changes.```

---
## ii) Property

_A_ **property** _is any **measurable characteristic** of a system._

Examples:  
Pressure (P), Temperature (T), Volume (V), Density (ρ), Internal Energy (U).

Properties depend only on the state, not on how the system reached that state.
there are:
[[Intensive properties]],[[Extensive properties]]

---

## iii) Process

A **process** is the transformation of a system from one state to another.

It represents the path followed during the change.  
Example: Isothermal process (T = constant).

---

## iv) Cycle

A **cycle** is a _series of processes_ after which the _system returns to its initial state._

`Initial state = Final state. ` 
Example: Heat engine cycle (like Carnot cycle).
